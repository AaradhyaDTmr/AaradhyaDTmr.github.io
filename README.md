# ADT.github.io

Personal portfolio website for Aaradhya Dev Tamrakar, an Electronics & Instrumentation Engineering student at Kathmandu Engineering College (KEC), Institute of Engineering (IOE).

## About

This is a GitHub Pages website showcasing my work as an electronics and instrumentation engineer, IEEE KEC Vice Secretary, and robotics enthusiast. The site features a modern, minimalist design with custom CSS and interactive elements.

## Features

- Responsive design optimized for all devices
- Custom cursor animation
- Smooth scrolling navigation
- Professional portfolio layout
- Electronics and robotics project showcase

## Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (Vanilla)
- Google Fonts (Cormorant Garamond, DM Mono, Instrument Sans)

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/AaradhyaDTmr/AaradhyaDTmr.github.io.git
   cd AaradhyaDTmr.github.io
   ```

2. Open `index.html` in your browser to view the site locally.

## Deployment

This site is automatically deployed using GitHub Pages. Any changes pushed to the `main` branch will be reflected on the live site at [https://aaradhyadtmr.github.io](https://aaradhyadtmr.github.io).
